# Facebook Ads Campaign Planning Project  
**Brand:** Sky Asia Tours  

## Project Type
Academic / Practice Project

## Project Overview
This project focuses on planning and structuring a Facebook Ads campaign to increase brand awareness and video views for a travel company.

## Objectives
- Increase brand awareness among target audiences  
- Drive video views for promotional content  
- Reach relevant travel-interested users  

## Work Done
- Selected campaign objective based on funnel stage  
- Defined target audience (age, gender, location, interests)  
- Created Facebook Business Page structure  
- Planned ad creatives and placements  
- Defined budget, schedule, and campaign duration  
- Identified key performance metrics  

## Tools Used
- Meta Ads Manager  
- Facebook Business Manager  
- Canva  

## Metrics Tracked
- Reach  
- Impressions  
- Video Views  
- Cost per Result  

## Outcome
- Developed a structured Facebook Ads campaign plan  
- Improved understanding of audience targeting and ad strategy
