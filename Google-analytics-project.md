# Google Analytics 4 (GA4) Setup & Analysis Project  

## Project Type
Academic / Practice Project

## Project Overview
This project focuses on setting up Google Analytics 4 (GA4) and analyzing website data to understand user behavior and performance metrics.

## Objectives
- Set up GA4 property  
- Track user activity and events  
- Analyze traffic sources and user engagement  

## Work Done
- Created GA4 property  
- Configured data streams  
- Analyzed audience reports  
- Reviewed acquisition and engagement metrics  
- Interpreted user behavior data  

## Tools Used
- Google Analytics 4 (GA4)  
- Google Tag Manager (basic)  

## Metrics Tracked
- Users  
- Sessions  
- Engagement Rate  
- Traffic Sources  

## Outcome
- Gained hands-on experience with GA4  
- Improved ability to analyze and interpret website data
